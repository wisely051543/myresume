The BAC Messaging Platform is a website system that importing assets and scheduling content to Sedna player for the BAC (Bank of America Corp) electric screen in Time Square.

For the requirement, we've developed the CSS animation web layer and playing video based on HTML5. This feature can combine multiple layer into a full screen version of HTML content, arranging the layers by dragging and dropping and scheduling final container showing time on calendar.

In this project, I responsible of front-end Javascript coding by framework Angular JS and CSS modification.
